Ways you can contribute, perfect for beginners!

Basic Python

Contribute by writing basic Python3 algorithms for various tools available on dcode.fr from the easiest ones like finding duplicates in a list to complex ones like QR code reader. Learn and diversify.

Python3 GUI via tkinter

Tkinter is a very simple and easy-to-learn module in Python3 that helps you design decent Graphical User Interface. Contribute to the looks and display of ENCODN by introducing frames for various tools, or by optimisng the already existing framework.

Graphic Design

ENCODN needs a logo. If you're a designer, you can contribute a logo to the repo :)
Or easier, lookout for releases of the app, and keep updating the sample image of how the GUI looks in the README.md file. That is super easy.
